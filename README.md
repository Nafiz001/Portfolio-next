## üìù Customization

### Update Personal Information

- **Contact Info**: Edit `components/Header.tsx` and `components/Footer.tsx`
- **About Section**: Modify `components/About.tsx`
- **Projects**: Update project data in `components/Projects.tsx`
- **Social Links**: Change URLs in `components/SocialSidebar.tsx`

### Change Colors

Edit `app/globals.css` to change colors.
